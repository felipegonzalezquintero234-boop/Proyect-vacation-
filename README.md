# Proyecto on vacation
    Este proyecto consiste en la creación de un sitio web básico para la empresa On Vacation, utilizando HTML y Bootstrap.

1. Creación de la página principal (index.html)
2. Creación de la página de servicios (servicios.html)
3. Implementación de un menú de navegación entre páginas
4. Inclusión de un título en cada página
5. Sección de bienvenida en la página principal
6. Botón de acceso a la página de servicios
7. Lista de servicios ofrecidos por la empresa
8. Uso de estilos con Bootstrap (CSS y JS)
9. Diseño básico responsive (adaptable a diferentes pantallas)
10. Inclusión de un footer en ambas páginas
Tecnologías utilizadas  
HTML5
Bootstrap 5
CSS (a través de Bootstrap)
